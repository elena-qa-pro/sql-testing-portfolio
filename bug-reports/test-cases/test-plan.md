# Test Plan

## Project
SQL Testing Portfolio Demo System

---

## Objective
Verify system functionality, database integrity, and API reliability.

---

## Test Scope

### Included
- Login module
- Payment module
- Database validation
- API endpoints

### Excluded
- Performance testing
- Security testing

---

## Test Approach

Testing will be performed using:

- Manual testing
- SQL validation queries
- API verification
- Negative scenarios
- Boundary value testing

---

## Test Environment
- Browser: Chrome
- OS: macOS
- Database: MySQL (simulated)
- API client: Postman

---

## Entry Criteria
Testing starts when:
- Build deployed
- DB accessible
- API endpoints available

---

## Exit Criteria
Testing ends when:
- All test cases executed
- Critical bugs fixed
- Regression passed

---

## Risks
- Incomplete test data
- Environment instability
- API failures

---

## Deliverables
- Test cases
- Bug reports
- SQL validation queries
- Test summary report
